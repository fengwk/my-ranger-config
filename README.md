# My Ranger Config

个人ranger配置。

# Installation

ranger

```
yay -S ranger
git clone git@github.com:fengwk/my-ranger-config.git ~/.config/ranger
```

特性

- 图标
- 图片预览：`yay -S ueberzug`
- 视频缩略图预览：`yay -S ffmpegthumbnailer`
- 代码高亮：`yay -S highlight`
- 压缩包预览：`yay -S atool`
- PDF预览：`yay -S poppler`
- 使用`ctrl+f`进行fzf搜索：`yay -S fzf`
